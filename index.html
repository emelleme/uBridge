mobiscroll by acidb

Customizable jQuery date and time picker for touch devices (Android, iOS)

A wheel scroller user control optimized for touchscreens to easily enter date and/or time. The control can easily be customized to support any custom values and can even be used as an intuitive alternative to the native select control (dropdown list). It is designed to be used on touch devices as an alternative to the jQuery UI date picker. The control is themable. You can easily change the appearance of if in CSS. It also comes with pre-defined, nice looking color schemes. By default the control renders for intuitive and easy usage, but you can also customize it's parameters (width, height,...). For more info see the wiki. See demo here: http://theacidmedia.net/demo.html More features and details to follow... Tested on iOS4, Android 2.2, Android 2.3, Chrome, Safari, Firefox. Please submit issues, and compatibility problems with other devices. If you like it, spread the word! Tweet like you never tweeted before :D
